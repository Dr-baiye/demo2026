# AI 对话工作台

面试笔试题

## 技术栈

- React 18 + TypeScript
- Vite 5
- Tailwind CSS
- framer-motion（页面过渡动画）
- lucide-react（图标库）
- react-router-dom（路由）

## 功能

- **对话页**：多智能体切换（6 种 AI 角色）、模拟流式输出、历史会话管理
- **设置页**：模型选择（预设 + 自定义）、推理参数调节、暗色模式
- **数据持久化**：localStorage 断电重连

## 运行

```bash
npm install
npm run dev
```
